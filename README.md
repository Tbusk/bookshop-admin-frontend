# Bookshop - Admin Page
A example admin page for a bookshop for SENG 400.

## Technologies Used
- React
- TypeScript
- Formik
- Yup
- PrimeReact (DataTable, Dialog, Icons, Calendar, MultiSelect)
- Hilla (AppLayout, Tabs)
- Bootstrap

This was a two-part project with the backend in Spring Boot. View the backend here -> https://github.com/Tbusk/bookshop-admin-backend
